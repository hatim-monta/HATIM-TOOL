# HATIM-TOOL
PYTHON SCRIPT FOR SCAN
Avant d’exécuter le script installer l’exigence « pyfiglet » :
$ pip install pyfiglet
$ git clone https://github.com/hatim-monta/HATIM-TOOL.git
$ cd HATIM-TOOL
$ python3 hatim-script.py ip/hostname
